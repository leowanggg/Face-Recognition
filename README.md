### Installation:

#### Systems Tested:

- Fedora 20

#### Requirements:

- OpenCV version 2.4 (Need FaceRecognizer Class)
- node.js and npm
- sails.js  
- [graphicsmagick](http://www.graphicsmagick.org)

#### Dependencies for sails.js:


- gm
- grunt
- sails-mongo
- ejs
- optimist
- opencv
- bcrypt
- crypto

### Start:

    $npm install [dependencies]
 
    $sails lift
